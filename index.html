Como usar:

Como Cadastrar?

É muito simples, basta passar o mouse sobre a opção cadastrar logo acima, na barra azul-escura; ao fazer isso, irá surgir uma bandeja de opções para que você possa selecionar o que deseja cadastrar no banco de dados. Selecionada a opção, basta preencher os campos de acordo com o que é solicitado. Como Consultar?

Para encontrar qual tabela você deseja consultar, é como encontrar o que você deseja cadastrar (a solução para esta questão foi apresentada no primeiro parágrafo); após selecionar o que você deseja consultar, será exibida uma tabela com todos os dados organizados do objeto selecionado. Note que é possível realizar uma pesquisa nessa tabela, filtrando-a dependendo do campo que houver: algumas tabelas podem ser filtradas com CPF, outras com o Código de Barras, também é possível pesquisar utilizando mais de um campo, isso ocorre na tabela de vendas, por exemplo.

Como Atualizar?

Para encontrar selecionar um objeto você já sabe, é só repetir o processo feito nos tópicos anteriores, selecionando a opção na qual estiver escrito "atualizar". Quando você escolher o objeto que deseja atualizar, será exibida uma tabela idêntica à tabela exibida quando é feita uma consulta, há somente uma diferença: haverá uma célula com um botão para que você possa selecionar o registro a ser atualizado, após selecioná-lo, será exibido um formulário semelhante ao formulário de cadastro, com a diferença que este estará preenchido com os dados originais daquele registro, basta fazer as alterações desejadas e clicar em enviar, por último aparecerá uma nova página com uma mensagem dizendo se a atualização foi feita com sucesso e um botão para retornar á página de atualização.

Alguns dados são impossíveis de serem atualizados por conta da estrutura do banco de dado. Também é possível que haja um erro dependendo do dado que seja colocado, por exemplo: se for inserido um número de telefone já existente na tabela de clientes, essa atualização não será feita, pois é impossível registrar o mesmo número de telefone em dois clientes. Mas não se preocupe, há diversas verificações de segurança em nosso sistema impedindo que qualquer erro o prejudique.

Como Deletar?

O processo é semelhante ao processo de atualizar. Ao clicar na tabela na qual deseja-se deletar determinados registros, aparecerá uma página com uma lista de todos os registros daquela tabela, é como na página de atualização, com a diferença que, o botão terá o formato de uma lixeira; para deletar basta clicar na lixeira respectiva do registro do qual você quer se livrar.

Assim como na atualização de registros, pode ser que ocorra um erro ao tentar deletar um registro em uso. Por exemplo, caso você queira deletar um cliente que participou de uma venda, não será possível. Somente será possível deletar esse cliente se deletar as vendas nas quais ele participa; esse princípio funciona para todos os casos de impossibilidade remoção de registro.
